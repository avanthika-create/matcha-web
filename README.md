A mock Matcha Café website designed and built using HTML, CSS, and JavaScript.  
It includes a landing page, interactive menu cards that display product details when clicked, and basic button interactions to create a café-style experience. 
Contains a menu, and a quiz to see what users would like to order. After taking the quiz, users can search for the item and they can locate what areas have the matcha product they pick.

For #horizons 
Design components: Minimal branding style that has a soft green theme.
